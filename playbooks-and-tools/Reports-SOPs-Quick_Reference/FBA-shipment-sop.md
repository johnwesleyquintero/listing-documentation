# SOP – Processing FBA Shipments for Smartwatches & Devices (SecuLife, STK, Emoji Kids)

**Objective:**
Ensure accurate recording, labeling, and submission of all device shipments to Amazon FBA, following proper sequencing, internal tracking, and compliance standards.

---

## 1. Pre‑Shipment Validation

1. Open **Excel master shipment file**
   ![SCREENSHOT: Excel master file](#)
   → Verify SKUs, quantities, and planned shipments match your internal tracking.

2. Log in to **TELEMOBILE Portal**: `https://portal.telemobile.me/bsindex.aspx`
   ![SCREENSHOT: Portal login](#)
   Navigate: `Reports → Ecom Tools → Amazon FBA SL → Shipments and Tracking`
   ![SCREENSHOT: Portal shipment table](#)
   → Confirm portal data matches Excel file.

3. Verify **Ship-From Address** and **Marketplace Destination**
   ![SCREENSHOT: Ship-From / Destination fields](#)

4. Check **FBA prep requirements** (batteries, fragile items, electronics)
   ![SCREENSHOT: Prep requirements checklist](#)

5. Record any deviations or partial shipment notes
   ![SCREENSHOT: Notes column](#)

---

## 2. Shipment Setup (Amazon Seller Central)

1. Log in to **Amazon Seller Central**
   ![SCREENSHOT: Seller Central login](#)

2. Navigate:
   `Inventory → Manage FBA Shipments → Send to Amazon → Start New Shipment`
   ![SCREENSHOT: Start New Shipment button](#)

3. Select **SKU(s) to ship**
   ![SCREENSHOT: SKU selection dropdown](#)

4. Determine **packaging type**:

   * Case-Pack (multiple identical units)
     ![SCREENSHOT: Case-Pack option](#)
   * Individual Units (single units, mixed SKUs allowed)
     ![SCREENSHOT: Individual Units option](#)

5. Set **box count and units per box**:

| Brand                 | Units per box | Max units per shipment |
| --------------------- | ------------- | ---------------------- |
| SecuLife / Emoji Kids | 50            | 150                    |
| STK                   | 100           | 500                    |

6. Rename workflow using **FBA shipment number**
   ![SCREENSHOT: Rename workflow field](#)

---

## 3. Packing & Labeling

1. Select **Package Details** → `Individual Units` if missing
   ![SCREENSHOT: Package Details selection](#)

2. Click **Ready to Pack**
   ![SCREENSHOT: Ready to Pack button](#)

3. Print labels carefully:

   * **FNSKU** for each individual unit
     ![SCREENSHOT: FNSKU print option](#)
   * **Box ID labels**
     ![SCREENSHOT: Box ID print option](#)
   * **Battery / Hazard labels** (if required)
     ![SCREENSHOT: Hazard label example](#)

4. Save labels to **OneDrive FBA folder**
   ![SCREENSHOT: OneDrive folder structure](#)

5. Rename each label according to sequencing number
   ![SCREENSHOT: Label rename example](#)

6. Take **screenshot of shipment cover page**
   ![SCREENSHOT: Shipment cover page](#)

---

1. Click **Pack Individual Units**
   ![SCREENSHOT: Pack Individual Units button](#)

2. For multiple boxes:

   * Confirm **box count and units per box**
     ![SCREENSHOT: Box summary](#)

   ![SCREENSHOT: Auto-populated fields](#)

## 5. Verify Shipment Details

## 10. Notes & Best Practices

* Minimum units per shipment: SecuLife 50, STK 100
* Maximum units per shipment: SecuLife 150, STK 500
* Always check box sizes and weights
* Maintain sequence numbers consistently
* Devices with batteries require correct hazardous labeling
* Screenshots/PDFs act as shipment cover reference
* Track carrier info post-shipment (SPD tracking, LTL PRO/BOL for pallet shipments)
