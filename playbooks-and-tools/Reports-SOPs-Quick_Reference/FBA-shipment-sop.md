SOP – Processing FBA Shipments for Smartwatches & Devices (SecuLife, STK, Emoji Kids)

Objective:
Ensure accurate recording, labeling, and submission of all device shipments to Amazon FBA, following proper sequencing, internal tracking, and compliance standards.

---

1. Pre-Shipment Validation

* Open **Excel master shipment file** [SCREENSHOT: Excel master file]
* Log in to **TELEMOBILE Portal**: [https://portal.telemobile.me/bsindex.aspx](https://portal.telemobile.me/bsindex.aspx)

  * Navigate: Reports → Ecom Tools → Amazon FBA SL → Shipments and Tracking [SCREENSHOT: Portal shipment table]
  * Confirm portal data matches Excel master file.
* Verify **Ship-From Address** and **Marketplace Destination** [SCREENSHOT: Ship-From / Destination fields]
* Check **FBA prep requirements** (batteries, fragile items, electronics) [SCREENSHOT: Prep requirements checklist]
* Record deviations or partial shipment notes [SCREENSHOT: Notes column]

---

2. Shipment Setup (Amazon Seller Central)

* Log in to **Amazon Seller Central** [SCREENSHOT: Seller Central login]
* Navigate: Inventory → Manage FBA Shipments → Send to Amazon → Start New Shipment [SCREENSHOT: Start New Shipment button]
* Select **SKU(s) to ship** [SCREENSHOT: SKU selection dropdown]
* Determine **packaging type**:

  * Case-Pack [SCREENSHOT: Case-Pack option]
  * Individual Units [SCREENSHOT: Individual Units option]
* Set **box count and units per box**:

| Brand                 | Units per box | Max units per shipment |
| --------------------- | ------------- | ---------------------- |
| SecuLife / Emoji Kids | 50            | 150                    |
| STK                   | 100           | 500                    |

* Rename workflow using **FBA shipment number** [SCREENSHOT: Rename workflow field]

---

3. Packing & Labeling

* Select **Package Details**; choose Individual Units if missing [SCREENSHOT: Package Details selection]
* Click **Ready to Pack** [SCREENSHOT: Ready to Pack button]
* Print labels carefully:

  * **FNSKU** for each individual unit [SCREENSHOT: FNSKU print option]
  * **Box ID labels** [SCREENSHOT: Box ID print option]
  * **Battery / Hazard labels** if required [SCREENSHOT: Hazard label example]
* Save labels to **OneDrive FBA folder** [SCREENSHOT: OneDrive folder structure]
* Rename each label according to sequencing number [SCREENSHOT: Label rename example]
* Take **screenshot of shipment cover page** [SCREENSHOT: Shipment cover page]

---

4. Confirm Packing

* Click **Pack Individual Units** [SCREENSHOT: Pack Individual Units button]
* For multiple boxes:

  * Confirm **box count and units per box** [SCREENSHOT: Box summary]
  * Ensure **labels match records** [SCREENSHOT: Label match confirmation]
* Verify Amazon auto-populates remaining shipment info [SCREENSHOT: Auto-populated fields]

---

5. Verify Shipment Details

* Check:

  * **Shipment ID** matches printed label [SCREENSHOT: Shipment ID field]
  * **SKU** is correct [SCREENSHOT: SKU check]
  * **Quantity & packaging** match master file [SCREENSHOT: Quantity verification]
* Adjust quantities if needed for partial shipments
* Click **Confirm shipment** [SCREENSHOT: Confirm shipment button]

---

6. Send Shipment Information via Email

* Prepare email using previous FBA template [SCREENSHOT: Email template]
* Update with: product name, SKU, units, packaging, shipment ID
* Attach:

  * All shipment labels [SCREENSHOT: Attach labels]
  * Screenshots / PDFs of shipment cover page [SCREENSHOT: Attach screenshots]
  * Barcodes [SCREENSHOT: Attach barcodes]
* Send to shipping team **after supervisor verification** [SCREENSHOT: Send button]

---

7. Update Internal Portal Records

* Navigate: FBA SL → Add shipment record [SCREENSHOT: Add shipment record]
* Enter shipment data: quantity, SKU, plan info (if applicable), date [SCREENSHOT: Data entry fields]
* Click **Save** [SCREENSHOT: Save button]
* Notes:

  * Devices without preloaded plans → Net SKU = 0
  * Devices with active plans → enter plan data

---

8. Notify Team

* Post to **FBA SecuLife / Emoji Kids group** [SCREENSHOT: FBA group message]
* Inform team shipment request email has been sent
* Bold critical info [SCREENSHOT: Bold text example]

---

9. Final Checks & Cleanup

* Confirm all labels and shipment info are correctly saved [SCREENSHOT: Saved files]
* Delete printed labels after shipment completion [SCREENSHOT: Delete labels]
* Ensure Excel master file, TELEMOBILE Portal, and email records are accurate [SCREENSHOT: Excel & portal verification]
* Archive screenshots & PDFs for audit reference [SCREENSHOT: Archive folder]

---

10. Notes & Best Practices

* Minimum units per shipment: SecuLife 50, STK 100
* Maximum units per shipment: SecuLife 150, STK 500
* Always check box sizes and weights
* Maintain sequence numbers consistently
* Devices with batteries require correct hazardous labeling
* Screenshots/PDFs act as shipment cover reference
* Track carrier info post-shipment (SPD tracking, LTL PRO/BOL for pallet shipments)
